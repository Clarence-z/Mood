## 📋 项目背景

> 这是我的**数据库课程大作业**，也是我**第一次尝试全栈开发**。作为一个零基础起步的学习项目，我从头开始学习了 SpringBoot、Vue 和前后端交互，在这个过程中**大量借助了大模型的帮助**。项目功能虽然还算完整，但肯定还存在很多不足和可以优化的地方。

### 项目特点：
- **学习性质**：这是我的第一个全栈项目，主要目的是学习和实践
- **AI辅助开发**：在编码、调试、文档编写等方面得到了大模型的很多帮助
- **完整功能闭环**：实现了从需求分析、设计、编码到测试的基本流程
- **待完善**：尚未接触生产部署，一些环境配置可能不准确或需要调整

## ✨ 功能特性

### 📱 用户系统
- 🔐 JWT令牌认证与自动刷新
- 👤 用户注册/登录/个人信息管理
- 📊 个人数据统计面板
- 🖼️ 头像上传与裁剪

### 📓 心情日记
- 😊 情绪评分（1-5分可视化选择）
- 📝 Markdown格式日记支持
- 📈 心情趋势图表（7天/15天）
- 🔍 日记历史检索与统计

### ✅ 智能待办清单
- 📅 任务创建/编辑/删除/完成/置顶
- 🎯 多条件高级查询（状态/优先级/时间范围）
- ⏰ 日历视图截止时间提醒
- 📊 数据统计面板

### 🤖 AI对话助手
- 💬 基于Dify配置的AI对话功能
- 📚 支持本地Ollama模型或商用API
- 🧠 上下文记忆与对话历史
- ⚡ 流式响应，实时打字效果

### 🏆 成就系统
- 🎮 游戏化激励体系
- 📊 基于用户行为解锁成就
- 🏅 成就分类方便扩展
